== Description ==

### Time Ago   

One click convert date and time to "time ago" (for example: 1 hour ago).

### The Features

* Just activate the plugin to display time ago.
* Post, Pages, Comments time ago support.
* Settings panel to translate the plugin easily (e.g. texts like: min, mins, hour, hours .. etc).

== Installation ==

### Installation

1. Upload 'time-ago' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Now plugin is ready, but if you want to translate texts or activate comments time ago, go to your wp-admin > Time Ago.
